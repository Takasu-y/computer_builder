# Usage
パーツを選択し、[add PC]ボタンをクリックでコンピュータを構築します。  
選択するパーツは下記の４つ。
* CPU
* GPU
* Memory Card
* Storage

選択した部品のベンチマークを元に、
-ゲーミング用/仕事用それぞれのパフォーマンス
-選択した部品リスト
を出力します。  
パーツのBrandやModelなど、どの順番に選択してもPCを構築出来ます。
